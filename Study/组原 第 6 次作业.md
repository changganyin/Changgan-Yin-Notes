1) a:MDR, b:IR, C:MAR, d:PC
2) 取指令: PC -> MAR -> Memory -> MDR -> IR
3) 存访问的数据通路: ACC -> MDR -> Memory
    取访问的数据通路: Memory -> MDR -> ALU -> ACC
4) LDA X 的数据通路为: X -> MDR -> Memory -> MDR -> ALU -> ACC
5) ADD Y 的数据通路为: Y -> MAR -> Memory -> MDR + ACC -> ALU -> ACC
6) STA Z 的数据通路为: Z -> MAR, ACC -> MDR -> Memory
